# Corporate Meal Subscription Analytics – Excel Project

This Excel-based data analytics project focuses on transforming raw corporate meal subscription data into meaningful business insights through structured data cleaning, analysis, and visualization techniques. The project demonstrates practical Excel skills used in real-world business reporting and decision-making scenarios.

## Key Tasks Completed

### Data Cleaning & Preparation
* Standardized date formats for consistency and accurate analysis
* Corrected inconsistent spellings and formatting issues in the dataset
* Applied data validation techniques to improve data accuracy and reliability
* Converted raw transactional data into structured Excel tables for efficient data management

### Data Analysis & Visualization
* Implemented freeze panes to improve dataset readability and navigation
* Created pivot tables to summarize and analyze revenue, quantity, and sales performance
* Developed pivot charts and interactive dashboards for visual reporting
* Applied conditional formatting to identify top-selling and lowest-selling food items
* Generated charts to uncover business trends and operational insights
* Performed daily revenue forecasting to support future planning and decision-making

## Skills Demonstrated
* Microsoft Excel
* Data Cleaning & Data Preparation
* Pivot Tables & Pivot Charts
* Data Visualization
* Business Insight Generation
* Revenue Forecasting
